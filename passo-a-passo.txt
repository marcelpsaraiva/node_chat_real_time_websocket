1) criar um diretorio
2) iniciar o npm (npm init)
3) npm install express@4.15.3 --save
4) npm install ejs@2.5.6 --save
5) npm install consign@0.1.6 --save
6) npm install body-parser@1.17.2 --save
7) npm install express-validator@3.2.0 --save
8) criar estrutura de diretório
- config
	- server.js
- app.js
- app
	- views
	- controllers
	- models
	- routes
		- index.js
	- public
		- css
		- images
		- js
9) no app.js, configurar para importar as configurações do servidor (server.js)
10) em server.js, importar as dependencias necessárias para renderizar os recursos no servidor
